# notes-app

Repo for my first node.js app for taking notes;

## List of commands:

#### To add a note
```node app.js add --title="sample title" --body="sample body"``` <br/>

#### To remove a note
```node app.js remove --title="sample title"``` <br/>

#### To list all notes
```node app.js list``` <br/>

#### To read a note
```node app.js read --title="sample title"``` <br/>
